# [MetaMask] Site with SSO

*使用 `MetaMask` 登入的[網站](/)*

## 專案目標

1. 試圖讓純 *html/javascript* 網站使用 *MetaMask Plugin* 登入, 並取得**用戶公開地址**.
2. 試圖讓純 *html/javascript* 網站設置用戶的 *MeataMask Plugin* 轉帳至**特定網路(Network)和地址(Address)**.
